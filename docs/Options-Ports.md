# kellerkinder.httpPort
Sets the http port for caddy.

_This has to be an int_

*_Example_*
```
kellerkinder.httpPort = 8080;
```

# kellerkinder.httpsPort
Sets the https port for caddy.

_This has to be an int_

*_Example_*
```
kellerkinder.httpsPort = 8443;
```
